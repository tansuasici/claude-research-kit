# Decisions (ADR-style)

> Record framing, scope, and methods decisions here — the manuscript analogue of
> architecture decision records. Claude reads this (Tier 3) before Protected-Claim
> changes. Append; never rewrite history.

## Template

### ADR-NNN: <short title>
- **Date:** <YYYY-MM-DD>
- **Context:** <what prompted the decision — a reviewer point, a scope question, a framing fork>
- **Options considered:**
  1. <option A — tradeoff>
  2. <option B — tradeoff>
- **Decision:** <what was chosen>
- **Consequences:** <what this commits the manuscript to; what it rules out>
- **Approved by:** <author>
